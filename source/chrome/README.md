## GitHub Code Review Chrome Extension

### Installation: http://bit.ly/18yLliL

### Development

```bash
$ coffee -wc .
```

### Usage

* Check "Development" mode in Extensions tab in Chrome
* Choose project's root directory in "Load unpacked extension" dialog
